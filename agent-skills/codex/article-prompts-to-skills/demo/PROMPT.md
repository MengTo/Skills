Present the result as an evidence-first workbench matching the structure of `demo/expected-output.md`: source prompts on the left, a transformation stage in the center, and skill-package cards on the right. Show the one-capability-per-skill decision, the shared package checklist, and a complete Minimal / Recreate / Remix prompt set for the pointer-lens example.

Keep `demo/index.html` standalone with inline CSS and JavaScript, no external dependencies, realistic content, semantic landmarks, visible focus states, and a responsive layout from 390px through 1440px.
